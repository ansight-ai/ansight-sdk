# Ansight Branding Guide

This guide defines the baseline branding tokens for Ansight based on the provided source assets:

- `branding/icon.png`
- `branding/logo.png`

- Logo: `branding/logo.png`
- Icon: `branding/icon.png`

## Core Colors

Extracted from image histograms of the source assets.

- `Ansight Orange`: `#FA431F` (RGB `250, 67, 31`)
- `Black`: `#000000` (from icon background)
- `White`: `#FFFFFF` (used with transparency in logo)
